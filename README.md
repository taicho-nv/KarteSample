# KarteSample

## Setup 
1. podで必要なパッケージをインストール
```
$ pod install
```

2. carthageで必要なパッケージをインストール
```
$ carthage update
```

3. KarteSample.xcodeworkspaceを開き、プロジェクトをビルドする
